01 · Sinem HQ A Claude operating system I run my whole business from +

An AI coworker with structure. A router sends every request to the right part of the business, a memory layer updates itself as the system learns, and a library of skills runs on its own. Notion sits underneath as the task and to-do layer.

02 · AI Second Brain Saved links become structured knowledge +

I save things on my phone all day. This pipeline catches them, transcribes them, files each one under the right topic, and distils it into notes I can actually use later. Built on Andrej Karpathy's capture, distil, link, index.

The capture pipeline

capture→ distil→ link→ index

Three phone-to-Slack inboxes feed it, one per source:

#instagram-capture #youtube-capture #articles-capture

Each clip files itself under the right topic, gets distilled into a source note, and links to what I already know. No manual filing.
